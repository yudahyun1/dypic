다운로드 문제를 해결하시겠습니까?

파일 다운로드에 문제가 있으신가요? 브라우저에서 모든 쿠키를 삭제해 보세요.
쿠키를 삭제하면 로딩이나 포맷 등 사이트의 특정 문제를 해결할 수 있습니다.

Chrome

  • 1. 컴퓨터에서 Chrome을 엽니다.
  • 2. 오른쪽 상단에서 더보기 > 설정 을 클릭합니다.
  • 3. 개인 정보 보호 및 보안 > 쿠키 및 기타 사이트 데이터를 클릭합니다.
  • 4. 모든 사이트 데이터 및 권한 보기 > 모든 데이터 삭제를 클릭합니다.
  • 5. 삭제 를 클릭하여 확인합니다.
Outlink

Microsoft Edge

  • 1.Edge에서 브라우저 창의 오른쪽 위 모서리에서 설정 및 기타를 선택합니다.
  • 2. 설정 > 개인 정보, 검색 및 서비스를 선택합니다.
  • 3. 검색 데이터 지우기 > 지금 검색 데이터 지우기에서 지울 항목 선택을 선택합니다.
  • 4. 시간 범위의 목록에서 시간 범위를 선택합니다.
  • 5. 쿠키 및 기타 사이트 데이터를 선택한 다음 지금 지우기를 선택합니다.

이제 선택한 시간 범위에 대한 모든 쿠키 및 기타 사이트 데이터가 삭제됩니다. 대부분의 사이트에서 로그아웃됩니다.

Outlink

Safari

Mac용 Safari 앱에서 Safari > 환경설정을 선택하고 개인 정보 보호를 클릭한 후, 다음 중 하나를 수행하십시오.

  • 1. 추적하는 사이트가 쿠키 및 웹 사이트 데이터를 사용하여 사용자를 추적하지 못하도록 차단하기: '크로스 사이트 추적 방지'를 선택하십시오.
    사용자가 추적하는 웹 사이트를 방문하고 상호 작용하지 않는 경우 쿠키 및 웹 사이트 데이터는 삭제됩니다.
  • 2. 쿠키 항상 차단하기: ‘모든 쿠키 차단’을 선택하십시오.
    모든 웹 사이트, 타사 및 광고업체는 사용자의 Mac에 쿠키와 기타 데이터를 저장할 수 없습니다. 일부 웹 사이트가 정상적으로 동작하지 않을 수 있습니다.
  • 3. 쿠키 항상 허용하기: ‘모든 쿠키 차단’을 선택 해제하십시오.
    모든 웹 사이트, 타사 및 광고업체는 사용자의 Mac에 쿠키와 기타 데이터를 저장할 수 있습니다.
  • 4. 저장한 쿠키와 데이터 제거하기: ‘웹 사이트 데이터 관리’를 클릭하고 한 개 이상의 웹 사이트를 선택한 다음, 제거 또는 모두 제거를 클릭하십시오.
    데이터를 제거하면 추적을 줄일 수 있지만 사용자를 웹 사이트에서 로그아웃하거나 웹 사이트 동작을 변경할 수도 있습니다.
  • 5. 쿠키나 데이터를 저장한 웹 사이트 보기: 웹 사이트 데이터 관리를 클릭하십시오.
